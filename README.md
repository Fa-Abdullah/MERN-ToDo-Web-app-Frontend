# MERN Todo Frontend

A modern React-based frontend for a MERN Todo application. The application provides a clean and responsive interface for managing personal tasks with secure user authentication, multilingual support, dark/light themes, and profile management. It communicates with the backend through REST APIs to provide a complete task management experience.

---

## Features

- User Registration & Login
- Secure Authentication
- Create, Update, Delete, and Complete Tasks
- Filter Tasks (All / Active / Completed)
- Clear Completed Tasks
- User Profile Management
- Dark & Light Mode
- Arabic & English Language Support
- Responsive User Interface
- REST API Integration with Backend

---

## Tech Stack

- React.js
- React Router DOM
- Axios
- Bootstrap 5
- CSS3
- JavaScript (ES6+)

### Authentication
- User Login

### User Profile

### User Experience
- Dark / Light Theme
- Arabic / English Interface

---

## API Integration

The frontend communicates with the backend using REST APIs through Axios.

Main operations include:

- User Authentication
- User Registration
- Fetch User Tasks
- Create New Task
- Update Task Status
- Delete Task
- Clear Completed Tasks
- Update User Profile
